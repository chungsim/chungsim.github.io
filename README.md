# 🎮 Game Developer Portfolio

> 유니티 기반 게임 개발을 중심으로 꾸준한 학습과 실전 경험을 이어가고 있는 게임 클라이언트 개발자입니다. 새로운 기술을 받아들이고, 문제를 구조적으로 해결하며, 프로젝트를 통해 성장하는 과정을 중요하게 생각합니다.




* 지속적 학습 기반의 성장
  
  학생 동아리, 학사 과정 및 6개월간의 부트캠프를 통해 기초부터 실전까지 체계적으로 학습하며, 스스로 부족한 부분을 찾아 보완하는 습관을 갖추었습니다.



* 프로젝트 경험을 통한 단계적 발전
  
  다양한 기능 구현과 협업 프로젝트를 수행하며 설계·구현·테스트의 전체 흐름을 직접 경험했습니다. 이를 통해 개발자로서의 역량을 단계적으로 확장해왔습니다.



* 피드백을 적극적으로 받아들이는 태도
  
  실무 개발자와 동료들의 의견을 학습의 기회로 삼아 발전해왔으며, 앞으로도 객관적 평가를 통해 더 나은 개발자로 나아가고자 합니다
